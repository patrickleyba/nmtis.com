CompanionCloud


CompanionCloud is a pet tracking web application that pet cremation or similar establishments can manage an efficient workflow. This SaaS application can be used as a tool for order processing in pet cremation process. CompanionCloud uses a status based approach for its tracking function and informs staff as to what stage of processing the pet is currently going through. Pet owners also receive SMS and EMAIL notifications updates in order to ensure their order needs have been met the entire process from start to finish.
